■ 公開方法（GitHub Pages）
1. GitHubの公開用リポジトリを開く
2. このフォルダ内のファイルとassetsフォルダを、そのままアップロード
3. Settings → Pages → Deploy from a branch
4. Branchをmain、フォルダを/(root)にしてSave
